# Instancia Municipal de las Mujeres - Tulancingo

Este es el sitio web oficial de la Instancia Municipal de las Mujeres de Tulancingo de Bravo, Hidalgo.

## Características
- Diseño institucional premium.
- 10 temas de interés interactivos.
- Violentómetro interactivo y descargable.
- Botón de pánico y recursos de ayuda.
- Optimizado para dispositivos móviles.

## Tecnologías
- HTML5 / CSS3 (Vanilla)
- JavaScript (Vanilla Web Components)
- Lucide Icons
- HTML2PDF.js

## Despliegue
Este proyecto está diseñado para ser desplegado en **Vercel** o cualquier servidor de archivos estáticos.
